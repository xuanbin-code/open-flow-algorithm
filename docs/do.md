# do - 后测循环

先执行一次循环体，再判断条件表达式，条件为真时继续执行。等同于其他语言中的 `do-while`。

## XML 语法

```xml
<do expression="条件表达式">
    <!-- 循环体，可包含任意代码节点 -->
</do>
```

## 属性

| 属性 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `expression` | String | 是 | 循环条件表达式，在循环体执行后判断 |

## 示例

```xml
<!-- 基本 do 循环（含占位符） -->
<do expression="condition">
    <more/>
</do>

<!-- 复杂条件 -->
<do expression="part1 &amp;&amp; (part2 || part3) &amp;&amp; (part4 || part5 || part6)">
    <more/>
</do>
```

## 视觉形状

六边形（准备框），形状与 `while` 相同，但位置在循环体之后。

## 与 while 的区别

| | while | do |
|------|-------|-----|
| 判断时机 | 循环体执行**前** | 循环体执行**后** |
| 最少执行次数 | 0 次 | 1 次 |
| 视觉位置 | 循环体上方 | 循环体下方 |
